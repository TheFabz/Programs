Visualizar uma página web ou outro recurso disponibilizado normalmente inicia ao digitar uma URL no browser ou seguindo (acedendo) uma ligação. Primeiramente, a parte da URL referente ao servidor web é separada e transformada num endereço IP, por uma base de dados da Internet chamado Domain name system (DNS). O browser establece então uma conexão TCP-Ip com o servidor web localizado no endereço IP retornado.
